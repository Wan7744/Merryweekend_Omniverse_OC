# Merryweekend_Omniverse_OC
즐주라는 인간의 옴니버스 세계관 캐릭터들을 한곳에 정리하고 이미지 파일 불러오도록 만든 외부 사이트다. 람쥐.

1. 여기 있는 캐자들 다 내 오리지널 캐릭터, 자캐들이다. 상업적으로 이용하면 죽여버리겠다.
2. github 아이디는 내가 대학교에서 사용하는 ID다. 이미지 불러오려고 퍼블릭으로 설정했는데, 교수님꼐 걸릴까봐 솔직히 두렵다.
3. 노션에 캐릭터들 관련 위키 만들려고 한건데 설마 이거 깃허브 걸릴까? 안 걸릴 것 같은데.
4. 아무튼 여기 캐자들은 커뮤에 사용하거나, 티알 때 사용하려고 모아둔 캐릭터들이다. 앞으로 더 추가될 가능성 있음.
5. 커피 마시고 싶다.
6. 다 만들고 나면 청리분들께 자랑해야지. 히히.
